**Install from GitHub**

From R console:

```R
remotes::install_github("Techtonique/learningmachine")
```

**Install from R-universe**

```R
install.packages("learningmachine", repos = c("https://techtonique.r-universe.dev", "https://cloud.r-project.org"))
```
